## Usage

### Start Game
```bash
npm run dev
```

### Create Room
1. Enter your name
2. Click "Create New Room"
3. Share room code with players

### Join Room
1. Enter your name
2. Enter room code
3. Click "Join Room"

### Game Controls
- **Host**: Start game, call numbers, reset game, enable auto-draw
- **Players**: Mark cards, chat, send reactions
- **All**: Change themes, toggle sound, view scoreboard

### Themes
Click colored circles to change theme (syncs to all players in online mode)

### Auto Draw
Host can enable auto-draw and adjust interval with slider

### Chat
Type messages or click emoji buttons for quick reactions
